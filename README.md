# Parkinsons-Disease-Diagnosis-from-Gait-analysis-using-Machine-Learning-and-Sensor-Data-
Machine Learning-based Parkinson's Disease Diagnosis and Severity Prediction using Gait Sensor Data.

## Overview
This project uses Machine Learning techniques to detect Parkinson's Disease and predict disease severity using gait sensor data.

## Features
- Data Preprocessing
- Feature Extraction
- Parkinson's Disease Detection
- Severity Prediction using UPDRS Scores

## Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest Classifier
- Random Forest Regression

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
